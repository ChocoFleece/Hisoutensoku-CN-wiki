                              # 打开游戏后任务栏有显示，但是桌面找不到游戏窗口，任务栏预览图为全黑

                              问题原因

                              窗口到屏幕外面了，窗口位置坐标和窗口大小被莫名其妙错误记录了（位置为负数，窗口大小异常）

                              # 方法一

                              方法一

                              关闭游戏，双击重置窗口的bat即可

                              # 方法二

                              方法二

                              关闭游戏，打开 th123/modules/WindowResizer/WindowResizer.ini
                              参考图片进行设置
