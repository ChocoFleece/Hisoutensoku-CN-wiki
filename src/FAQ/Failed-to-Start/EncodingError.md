::: info 非中文系统开游戏时弹窗 Sokulobby Eoor

![非中文系统开游戏时弹窗 Sokulobby Eoor，报错弹窗如图，Windows区域设置为日文](https://bu.dusays.com/2024/01/21/65acc84cb5413.webp =400x)

非中文系统开游戏时弹窗 Sokulobby Eoor，是因为游戏系统语言不是简体中文，而游戏路径内包含中文汉字
::: tip 解决方法

1、改回系统语言为简体中文

2、可以将游戏路径改成全英文


:::
