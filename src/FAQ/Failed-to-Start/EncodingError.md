## 一、开游戏时弹窗 Sokulobby Eoor
::: info 

![图片加载中... =400x](https://bu.dusays.com/2024/01/21/65acc84cb5413.webp "非中文系统开游戏时弹窗 Sokulobby Eoor，报错弹窗如图，Windows区域设置为日文")

开游戏时弹窗 Sokulobby Eoor，很可能是因为游戏系统语言不是简体中文，而游戏路径内包含中文汉字
::: tip 解决方法

1、改回系统语言为简体中文

2、可以将游戏路径改成全英文


:::
