### 对战前初步检查自己的网络延迟是否稳定，WIFI打则解释和倡议

:::: details
:::: info 

![](https://bu.dusays.com/2024/07/25/66a1b9fb7b456.png "任务栏右下角，WIFI图标")

接网线的有线网络的延迟波动一般最多只有 5ms，可以忽略不计
而 WIFI、热点等无线网络的延迟波动好的会有小于 20ms 的较稳定情况，差的就会有上百 ms 的波动，打则或者别的游戏就会出现卡顿等情况

**为了避免给对手和自己造成不良游戏体验，建议先自己初步评测自己的网络波动情况，然后再决定要不要玩游戏**

对于WIFI打则玩家，不管你是否稳定，都建议在名字上标注WIFI等字眼告诉别人，并不需要感到任何不好意思或羞耻或者卑微，因为我们也并不希望在不良网络环境扫大家打游戏的兴，很多时候场地并没有网线可以使用。

**一般建议WIFI玩家自己建主**，标明并等待愿意玩WIFI对局的人加入
当然你也可以主动加入别人，但是请在进入别人IP之前，先向对方**表明自己WIFI的ping波动情况并询问对方意见**，同意了再进
==如果你故意或无意间隐瞒自己网络不稳定的情况，而和别人对战导致别人遭遇预期之外的不良游戏体验，那就是你的不对了==

别人可以选择不和WIFI玩，这是个人的选择权力，你也无需且不应该因为别人看你是WIFI就不和你玩而感到受到歧视或者卑微或者愤懑等。
但是你可以向大家**展示你的ping波动情况**并邀请他人尝试和你打一两局看看到底卡不卡，如果不卡那双方愿意继续玩就皆大欢喜，如果卡了那是否说明网络确实看对方的意见，对方拒绝继续玩的话那就到此为止你换个人玩，**互相尊重理解**。

两个WIFI一起玩的话是会叠加不稳定程度的，所以不良游戏体验的发生频率可能会更高，依旧是**互相尊重理解**的原则，没法打或者不想打了那就不打，干点别的事。

::: tip 关于网络延迟大小和网络延迟波动的误区澄清
- 打则黄标卡顿等问题的主要关键就在于**网络延迟波动**，而不是**网络延迟大小**

- 网络延迟大小是ping的时间，过大一般表明距离很远，如跨国到欧美。仅讨论国内的话，在新则回滚的帮助下，从最南端海南到最北端黑龙江的**稳定**的网络延迟下，**不是影响游戏体验的主要原因**
如稳定保持在60ms延迟，不影响游戏体验

- 网络延迟波动则是很多个ping包时间之间的起伏波动情况，波动幅度很大说明网络稳定性差，卡顿的可能性高，**是影响游戏体验的主要原因**
如一般是20ms延迟，但是经常性波动到几百上千ms的延迟，在这段波动时间内就会发生游戏内 ==正中央下方数字被黄标== 的卡顿情况

:::



::: details 通过ping来初步评测自己的网络波动情况
按下 `Win+R` 打开运行或直接打开cmd，粘贴 `cmd /k ping 8.8.8.8 -t` 后回车，稍微等待一会，
延迟有上下 20ms 左右的波动都属于正常情况。

**也可以检查游戏对局内 ==正中央下方数字网络延迟== 有没有一直在跳变导致偶尔黄标，过于频繁会导致游戏体验不好**

### **使用 Swarm 等中转 IP 在大厅里联机**
<!-- #region SwarmUsage -->

通过中转服务器，中转数据包的方式来帮助联机，只需要建主方打开 Swarm 软件即可，对连接方无要求


- 视频教程 —— [**😎 新人指路 2024 五月 “新则” 使用向导**](https://www.bilibili.com/video/BV1gZ421s7yF/)

- 游戏整合包内自带即有 Swarm-new，单独下载方式见  [123网盘（根目录/其他非想天则资源/备用联机工具）](https://www.123pan.com/s/scmzVv-Z3h6v.html)

:::: info 联机教程 - Swarm 使用说明

1. 点“获取服务器列表”（一般选择自己延迟最低的就行，[**绕远路时综合双方的列表进行选择**](/Beginners/BeforePlaying.html#提醒注意使用-swarm-不要绕远路了-按照流程选择适合自己的中转节点)）

2. 点“中转”可以得到一串 IP，然后点“复制”就自动帮你复制好了，可以发给别人了

4. 如果你想在大厅建主等待别人加入，可以点击“IP 用于大厅”的按钮；
重新点了“中转”刷新 IP 的话，也记得要再次点击“IP 用于大厅”，不需要重开游戏或重进大厅

1. 中转后点“端口管理”，可以查看自己和对手的网络延迟情况
红线是自己距离服务器延迟，不是总和。
大幅度波动说明可能是wifi无线网络，游戏内也会有波动风险

### 提醒注意使用 swarm 不要绕远路了，按照流程选择适合自己的中转节点
::: details

**如果你发现下方的网络延迟超过40，很有可能是绕了远路，此时联机双方两个人可以都截图发下自己的完整节点列表的延迟，然后选择相加延迟最小的中转服务器**

1、如果支持 公网IP直连 或 [**AutoPunch 直连**](/FAQ/Play/AP_NOT_Supported.html) 或 [**IPV6 直连**](/mods/AdvancedMods/IPv6Mod.html)，那就最好，不需要走 Swarm 中转

2、 上海、广东、北京、四川这四个节点围成的圈。处于圈上以及圈外部的人，选择列表上最低延迟的节点建主，就一定是最优节点。推荐自己发IP建主，让别人进你的IP，就不会发生绕远路的情况；自己进别人的IP也建议挑选符合你最优节点的IP进，不然可能会绕远路增大延迟

![图片加载中... =900x](https://bu.dusays.com/2024/10/31/6723a6cadae9c.jpg "圈外，直接选择列表上最低延迟的节点建主")

3、 位于围成的圈内，但是入圈内范围不大的人，也可以选择列表上最低延迟的节点建主，虽然会稍微绕一点远路，但是不多，可以接受

![图片加载中... =900x](https://bu.dusays.com/2024/10/31/6723a75f7eb3a.jpg "入圈内范围不大，直接选择列表上最低延迟的节点建主")

4、 再深入圈内的，可能会有遇到两个不同方位的节点的延迟都差不多的情况，即处于两个节点的中间地带，此时最不适合自己发IP建主。最好先看看别人发的IP和问问别人的列表延迟，否则就很可能会发生绕远路的情况

![图片加载中... =900x](https://bu.dusays.com/2024/10/31/6723a92fba9da.jpg "绕远路的错误案例")


::: tip 服务器所在地区：

- `124.223.180.38` —— 上海（东部）
- `8.134.214.80` —— 广东广州（南部）
- `47.108.225.175` —— 四川成都（西部）

### 成功进入对战，选人界面
::: info 成功进入对战，选人界面
- 进入后，请在QQ群内回复别人，让别人知道是在和谁玩，便于后续交流

- 可在选人界面按C，能且只能修改卡组

- 若不想继续打了，可以在群里at对手，或者在选人界面连续按X发出框框的声音，表示Last，最后一局

- 如果发现对手一直没反应，最好查看下QQ群消息和大厅聊天框，也许他正在那边打字


:::

::: info 进入对局后

- 如果发现对手一直没反应，请理解并耐心等待，他可能身边有点急事需要离开一会儿；
可以查看下 QQ 群消息和大厅聊天框，也许他正在那边打字

- 如果发现自己和对手实力悬殊（对手太强或者太新），可以在 QQ 群或聊天框内交流是否还要继续玩

- 最好不要无端故意中途 ESC 结束对局，会导致无法保存 Rep，
若有紧急情况要离开可以在事后 QQ 群或者聊天框通知对手
  > 2023.11.30 Hagb 制作了 SaveRep mod 可以在异常情况下紧急保存 Rep，<br>虽然 ESC 也可以保存 Rep 了但是依然不提倡“节省时间”而在对局时 ESC

- 打完结束后可以按两下X框框两声，并回复“谢谢指教”，“谢则”之类的话感谢对战

### **注意正中央网络延迟，并调整左下角延迟帧数以获得更佳游戏体验**

- 使用 `大键盘的加减号`(在 Backspace 键的左边) 调整左下角自己的输入延迟，默认为 2 即可，
若是跨国对战等遭遇大量回滚（高延迟），可尝试调至 d3 d4 看看能不能接受

- 当自己电脑性能不佳时，右下角帧数会变红

- 若因网络问题导致回滚范围超过最大范围，
则游戏画面会因暂停而表现为类似掉帧的现象时，中央回滚数字会变黄

::: details 延迟数值与左下角输入延迟之间关系的详细解释
假设游戏内正下方的数字显示 80，就相当于总共延迟 5 帧，此时

**1P 设置左下角 d1，就选择延迟 1F（1P 的指令输入像练习模式一样流畅），但是就需要 4F 的回滚帧（1P 画面里的对手发生回溯与瞬移的情况频发）；**

**2P 设置左下角 d3，就选择延迟 3F（2P 输入的指令过 3F 才会实际输入游戏内），不过只需要 2F 的回滚帧（2P 画面里的对手基本不会发生回溯与瞬移）**

### 确保电脑不超负荷运行

::: caution 注意右下角帧数应该为62，而不是60
- [**NVIDA 或 AMD 等显卡驱动设置里锁了最大帧率为 60 帧**](/FAQ/Play/LagPlaying.html)

- 当自己电脑性能不佳时右下角帧数会变红色

- 对局卡顿（网络原因或对手电脑问题）时中央回滚数字会变黄色

-----

- 可能导致电脑性能不佳的原因：
  - 直播/录屏的配置过高
  - 开启了省电模式
  - 没插电源
  - CPU 过热降频
  - 正在进行杀毒扫盘等
  - 正在编码/导出/压制视频
  - 显卡驱动有问题
:::
