### 开始游戏前，设置好你的机签名（Profile）、键位、卡组
::: info 
- <font color=red size=4><strong>在练习模式选人界面按 C键 打开菜单</strong></font>

- 若要修改机签名，直接去打开你的 `th123` 游戏文件夹，找到 `profile` 文件夹，然后把 `profile1p.pf` 修改为你想要的文件名即可，如修改为 `入则新人.pf`

- **先选择好机签，右上角变化后，再去设置 1P 卡组和按键**

- 机签名字最好是可以方便在 QQ 等地方找到你的名字，不要设置低俗无礼或者无意义或者不好打出来的名字

- 可以将自己的游戏水平标注，有助于寻找旗鼓相当的对手

![右上角就是选择好了的，正在使用的机签名，如图中是 Tstar00](https://bu.dusays.com/2024/03/27/6603aa1341843.png =600x)

:::

### 确保网络连接稳定

::: caution
==**显卡驱动有问题可能会导致游戏卡顿**==

==**低电量（节能模式）或高占用 CPU 内存可能会导致游戏卡顿**==

[**跳转链接 —— 为什么联机很卡**](/FAQ/Play/LagPlaying.html)
:::

::: info 检查网络波动情况

按下 `Win+R`，复制粘贴 `cmd /k ping baidu.com -t` 并回车，等待一会儿，
延迟波动范围上下 15ms 左右都是正常情况，可以接受

**也可以检查游戏对局内正中央下方数字有没有一直在跳变，若有则很可能说明有人是 Wifi，网络不稳定**

![像这样延迟波动不大则可以认为是稳定的，进行游戏不会卡顿](https://bu.dusays.com/2023/08/22/64e4a1d8cbfce.png =500x)

### **使用 Swarm 等中转 IP 在大厅里联机**

::: tip Swarm 介绍

**Swarm 通过中转服务器，中转数据包的方式来帮助联机，只需要建主方打开 Swarm 软件即可，对连接方无要求**

:::

:::: info 联机教程 - Swarm使用说明

1. **双击 `[ 启动 ] 游戏+Swarm联机中转.bat` 可以一键打开非想天则游戏和 Swarm 两个程序**

![](https://bu.dusays.com/2024/02/27/65dcc714d6ec0.png =250x)

2. **点“获取服务器列表”**（一般选择自己延迟最低的就行，绕远路时综合双方的列表进行选择）

3. **点“中转”可以得到一串 IP，然后点“复制”就自动帮你复制好了，可以发给别人了**

4. **如果别人进不来，请尝试再次点击“中转”，然后“复制”新的 IP 发给别人**

5. **如果你想在大厅建主等待别人加入，可以点击“IP 用于大厅”的按钮； <br>重新点了“中转”刷新 IP 的话，也记得要再次点击“IP 用于大厅”，不需要重开游戏或重进大厅**

6. **提示：在大厅建主的同时，你也可以往群里发 IP，别人也可以通过 IP 的方式连接你**

![**swarm 和则之间不存在打开的先后顺序，哪个先开后开都无所谓**](https://bu.dusays.com/2024/02/26/65dcaf689e6a6.jpg =240x)

- **中转后点“端口管理”，可以查看自己和对手的网络延迟情况**

### 注意正中央网络延迟并调整左下角延迟帧数

:::: info 进入对局后

- **如果发现对手一直没反应，请理解并耐心等待，他可能身边有点急事需要离开一会儿；可以查看下QQ群消息和大厅聊天框，也许他正在那边打字**

- **如果发现自己和对手实力悬殊（对手太强或者太新），可以在QQ群或聊天框内交流是否还要继续玩**

- **最好不要无端故意中途ESC结束对局，会导致无法保存Rep，若有紧急情况要离开可以在事后QQ群或者聊天框通知对手**

- **2023.11.30 Hagb制作了SaveRep的mod可以在异常情况下紧急保存rep，虽然ESC也可以保存Rep了但是依然不提倡“节省时间”而在对局时ESC**

- **打完结束后可以按两下X框框两声，并回复谢指教，谢则之类的话感谢对战**


::: tip 注意正中央网络延迟并调整左下角延迟帧数

**使用 大键盘的加减号(在 Backspace 键的左边) 调整左下角 delay 值，调整规则可参考下图**

**简单来说就是：从d1开始，若发生频繁回滚现象则增大输入延迟，直到均衡满意**

Giuroll 0.6 系列版本，增加“单方回滚”特性：对战双方各自按大键盘加减号（Backspace退格键左边的那个），来调整自己的左下角延迟帧数，互相不影响对方

![giuroll 0.6.3+ 数据说明](https://bu.dusays.com/2024/01/21/65acd056779ab.webp =600x)

![根据正中央的网络延迟调整左下角的输入延迟帧数](https://bu.dusays.com/2024/01/21/65acba0768ae5.webp =700x)

:::
::: details 延迟数值与左下角输入延迟之间关系的详细解释
假设游戏内正下方的数字显示 40，说明双方网络延迟是 80ms 左右，就相当于总共延迟 5 帧，此时

**1P 设置左下角 d1，就选择延迟 1F（1P 的指令输入像练习模式一样流畅），但是就需要 4F 的回滚帧（1P 画面里的对手发生回溯与瞬移的情况频发）；**

**2P 设置左下角 d3，就选择延迟 3F（2P 输入的指令过 3F 才会实际输入游戏内），不过只需要 2F 的回滚帧（2P 画面里的对手基本不会发生回溯与瞬移）**
:::
